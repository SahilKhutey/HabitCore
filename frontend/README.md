# HabitHero Frontend 📱💻

One codebase. Three platforms. Zero friction.

## 🚀 Cross-Platform Strategy
Built with **React Native (Expo)** to target:
- **Android**: via Play Store (.aab).
- **iOS**: via App Store (.ipa).
- **Web/Windows**: via PWA (Microsoft Store).

## 🛠️ Key Technologies
- **Expo Router**: File-based routing for seamless navigation.
- **Zustand**: Lightweight global state for user profiles and preferences.
- **Lucide-React-Native**: Consistent, professional iconography across all platforms.
- **Moti / Reanimated**: High-performance animations for a premium feel.

## 🎨 Design System
- **Dark Mode Default**: Optimized for focus and eye-comfort.
- **Vibrant Accents**: High-contrast colors for primary actions.
- **Premium Themes**: Gold/Black aesthetics for the Hall of Heroes.

## 📦 Build Commands
- `npm run start`: Local development.
- `npm run build:web`: Export PWA static bundle.
- `eas build --platform [android|ios]`: Trigger EAS mobile builds.
